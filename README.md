# Java Projects Collection

This repository contains three Java mini-projects developed using Core Java concepts such as OOP, ArrayList, Exception Handling, Loops, Conditional Statements, and File Handling.

---

# 1. SmartStay Hotel Management System

## Project Description

SmartStay is a console-based Hotel Management System developed in Java. The system helps manage hotel room bookings, guest details, billing, room availability, and waitlists.

## Features

* Admin Login System
* Guest Check-In & Check-Out
* Room Management
* Room Types & Pricing
* Billing System with GST
* Guest Search
* Waitlist Management
* Date Handling
* Exception Handling
* Menu-Driven Interface

## Technologies Used

* Java
* OOP Concepts
* ArrayList
* Queue
* LocalDate API
* Exception Handling

## Default Login

Username: admin
Password: 1234

## How to Run

```bash
javac SmartStayHotelManagementSystem.java
java SmartStayHotelManagementSystem
```

* Passenger Details Management
